## Hyolmo Database Engineering Project

This repository demonstrates PostgreSQL database engineering skills through
analysis and improvement of a real Django ORM–generated schema.

### Scope of Work
- Extracted production database schema using pg_dump
- Improved schema design through normalization and correct data types
- Designed indexing strategy for query performance
- Enforced relational integrity using foreign keys
- Version-controlled database changes using Git

### Files Overview
- 01_extracted_schema.sql — Original ORM-generated schema (baseline)
- 02_improved_schema.sql — Normalized tables and schema improvements
- 03_indexes.sql — Indexing strategy for performance
- 04_relationships.sql — Foreign key constraints and relationships

### Technologies
- PostgreSQL
- SQL
- Django ORM (schema source)
- Git & GitHub
